# CO2 Detection System

An IoT project using Node-MCU, DHT22 Sensor, MQ7 Sensor, with GUI built using Flutter and Firebase.

## Hardware Part

This project using Node-MCU, DHT22 Temperature and Humidity Sensor, MQ7 CO_2 Sensor
